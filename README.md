# MeterNumberPickerLibraryDemo

Android Meter Number Picker Library Demo

App created in Android Studio 3.0.1 and tested on a Genymotion Google Nexus 5X Android Emulator then on a Motorola Moto X Developer Edition Android phone.

To use this app the user just clicks the 'Meter Number Picker Library Demo' app icon and they are taken right into the app. 
They are greeted with a title ('Meter Number Picker Library Demo')on the top. Then in the main body two different sized number
meters in different colors. On the bottom are two buttons that when clicked display a Toast message.

As the user flips through the meter they can then click the button which corresponds to the correct counter (top or bottom) and 
a Toast message calling out the counter called and the number selected. This demo uses the library 
<a href="https://github.com/alexzaitsev/meter-number-picker">Meter Number Picker</a> to create the meter effect.
